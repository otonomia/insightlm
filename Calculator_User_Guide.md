# Calculator User Guide

This is a placeholder guide demonstrating the output format produced by
`calculator_agent.py`. Run the agent to regenerate this document with fresh
observations from your system's calculator.

## Basic Operations
- Addition: `2 + 2 = 4`
- Multiplication: `5 * 3 = 15`

The agent automatically runs a few sample calculations and records the display
using OCR. The collected logs are then summarized into this Markdown guide.
